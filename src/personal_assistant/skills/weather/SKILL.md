# Weather（天气查询）

> 查询指定城市的实时天气和未来预报，帮助用户了解天气情况以决定出行安排。

## 使用场景

- 用户询问今天/明天的天气
- 用户计划出行，需要了解目的地天气
- 用户想知道是否需要带伞、穿什么衣服

## 使用步骤

1. 用 bash 工具执行天气查询脚本，传入城市名：

```bash
python "<技能文件夹>/weather.py" "城市名"
```

注意：`<技能文件夹>` 的实际路径已在技能内容开头的 `[技能文件夹: ...]` 中给出，请替换为实际路径。

2. 脚本会返回格式化天气信息（温度、湿度、天气状况、风力等）。
3. 根据天气信息为用户提供建议（如带伞、添衣等）。

## 示例

```bash
# 查询北京天气
python "/path/to/skills/weather/weather.py" "北京"

# 查询上海天气
python "/path/to/skills/weather/weather.py" "上海"

# 查询英文城市名
python "/path/to/skills/weather/weather.py" "Tokyo"
```

## 注意事项

- 使用 wttr.in 免费天气 API，无需 API key
- 支持中文和英文城市名
- 如果查询失败，可尝试使用城市的英文名
