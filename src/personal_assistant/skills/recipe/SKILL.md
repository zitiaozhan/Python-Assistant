# Recipe Finder（菜谱查询）

> 根据菜名或食材搜索菜谱，提供详细的制作步骤和所需材料，帮助用户解决"今天吃什么"的难题。

## 使用场景

- 用户询问"今天吃什么"
- 用户提供食材，想知道能做什么菜
- 用户想学习某道菜的做法

## 使用步骤

1. 用 bash 工具执行菜谱查询脚本：

```bash
python "<技能文件夹>/recipe.py" "菜名或食材"
```

注意：`<技能文件夹>` 的实际路径已在技能内容开头的 `[技能文件夹: ...]` 中给出，请替换为实际路径。

2. 脚本返回菜谱信息（菜名、食材、步骤）。
3. 为用户总结菜谱要点，可提供烹饪建议。

## 示例

```bash
# 查询宫保鸡丁做法
python "/path/to/skills/recipe/recipe.py" "宫保鸡丁"

# 查询番茄能做什么菜
python "/path/to/skills/recipe/recipe.py" "番茄"

# 查询红烧肉
python "/path/to/skills/recipe/recipe.py" "红烧肉"
```

## 注意事项

- 使用网络搜索获取菜谱信息
- 如果查询失败，可尝试简化关键词
- 返回的菜谱数量为 5 条，避免信息过载
