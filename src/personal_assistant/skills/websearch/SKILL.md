# Web Search

> 使用必应（Bing）搜索引擎搜索网络信息，获取实时新闻、最新资讯或任何需要联网查询的内容。当用户需要搜索网络、查找新闻、获取最新信息时使用此技能。

## 使用场景

- 用户询问最新新闻或当前事件
- 需要查找你训练数据中可能没有的实时信息
- 用户明确要求搜索某个话题

## 使用步骤

1. 用 bash 工具执行搜索脚本，传入搜索关键词：

```bash
python "<技能文件夹>/search.py" "搜索关键词"
```

注意：`<技能文件夹>` 的实际路径已在技能内容开头的 `[技能文件夹: ...]` 中给出，请替换为实际路径。

2. 脚本会返回搜索结果（标题、链接、摘要），最多 10 条。
3. 阅读搜索结果，为用户总结关键信息。
4. 如果结果不够，可以调整关键词重新搜索。
5. 与用户同一个对话轮次中相同任务的搜索不要超过3次，最多搜索3次后就要继续下一步处理。

## 示例

```bash
# 搜索马斯克新闻
python "/path/to/skills/websearch/search.py" "马斯克 最新新闻"

# 搜索技术文档
python "/path/to/skills/websearch/search.py" "Python asyncio tutorial"
```

## 注意事项

- 搜索脚本使用标准库实现，无需额外依赖。
- 搜索结果取决于 Bing 返回的内容，可能受网络环境影响。
- 如果搜索失败，可尝试简化关键词或稍后重试。
